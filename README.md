# 🌍 Country Info App

A live web app that fetches real-time country data 
using a public REST API and displays it instantly.

## What It Does
- User types any country name
- App fetches live data from REST Countries API
- Displays flag, capital, population, currency, language
- Handles errors and loading states

## Tech Used
- HTML, CSS, JavaScript
- REST Countries API (restcountries.com)
- fetch() with async/await
- DOM manipulation

## Concepts Practiced
- API calls using fetch()
- JSON parsing
- async/await error handling
- Dynamic DOM updates
